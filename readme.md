
<h1>Incubator STM Code</h1>
<p>This is the primary code for the Infant Incubator utilizing the STM Microcontroller</p>

<h2>Installation</h2>

To install and run the code run:


execute command:
git clone https://github.com/Innovative-Global-Solutions/Infant_Incubator_L152RE.git

Open STMCubeIDE > navigate to file > select "Open Project from File System" > Directory > Select "Infant_Incubator_L152RE" > finish

Attempt to build the code using the hammer icon

<h2>Adding features</h2>

To add features create a branch -> make changes -> push branch to remote -> open pull request and get a review
